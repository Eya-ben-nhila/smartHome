🏠 Home Assistant Setup

Configure your Home Assistant connection to enable real-time device control and monitoring

📖 How to get your Home Assistant URL:

1. Open Home Assistant in your browser

2. Look at the URL in your browser's address bar

3. Use that exact URL (including http:// or https://)

🔑 How to get your Access Token:

1. In Home Assistant, go to: Profile → Long-Lived Access Tokens

2. Click Create Token

3. Give the token a name (e.g., "smart-home-web")

4. Copy the generated token

5. Paste the token in the field above
