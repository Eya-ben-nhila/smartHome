🔧 Troubleshooting Tips

  • URL Format: Use http://homeassistant.local:8123 (no /api/ at end)
  • Access Token: Generate in HA → Profile → Long-Lived Access Tokens
  • Network: Make sure you're on same network as Home Assistant
  • Browser: Try Chrome or Firefox if Safari fails
  • HTTPS: Use http:// for local connections, https:// for remote
  • Firewall: Allow port 8123 through firewall
